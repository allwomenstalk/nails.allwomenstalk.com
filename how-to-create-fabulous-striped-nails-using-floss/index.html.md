# How to Create Fabulous Striped Nails Using Floss ...

You don't need to buy fancy tools in order to make your nails look amazing. If you want to make stripes across your nails, all you need is some dental floss!

Start by applying a base coat in any color that you'd like. Then you should grab the color that you'd like to add to your nails and press your floss against the polish, and then against the nail itself. Add as many stripes as you'd like until you're done!
